Welcome to MyAssist Get natural language processing responses to your queries with our personal assistant app. Conversational Tone MyAssist generates human-like responses to answer your questions in a lucid and conversational way. Microphone Functionality Instead of typing, you can send voice commands to MyAssist. Language Functionality MyAssist can reply to you in different languages if you record commands in that language. Chat History MyAssist saves the chat history of each user, allowing you to refer to previous messages in the conversation. Request Assistant Permissions If you create an account, you'll need to request permission to use the personal assistant app. The admin has the power to approve this request and allow you to log in. GPS Integration You can send location-based queries like "suggest some restaurants at my location," and MyAssist will fetch your location from GPS and send the query to the personal assistant. Clear Chat History You can send the command "clear the chat," and MyAssist will clear the chat history. Technology Stack React Native Used for mobile client app development Angular Used for web front-end Spring Framework Spring Boot is used for the backend. (Java) MySQL Used for the database Open AI Used for the API Google API Google API is used to fetch textual address of the user when we send it the GPS coordinates. Sensors MyAssist uses sensors such as the microphone and GPS to provide a more personalized and interactive experience.
